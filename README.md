# random-image

A utility library with common helper functions.

## Installation

```bash
npm install random-image
```

## Usage

```typescript
import { add, isEven, toSlug } from 'random-image';

console.log(add(1, 2)); // 3
console.log(isEven(4)); // true
console.log(toSlug('Hello World')); // hello-world
```

## Features

- **add**: Add two numbers.
- **isEven**: Check if a number is even.
- **toSlug**: Convert a string to a URL-friendly slug.
